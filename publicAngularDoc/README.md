# AngularJS to Angular use @angular/cli
## 安装以及初始化(别的内容单独写成.md文件了)
`nodejs v6.x.x(6.10.0)`  
`npm v3.x.x(3.10.10)`  
使用淘宝npm镜像
```text
npm install -g cnpm --registry=https://registry.npm.taobao.org
```
安装angular-cli
```text
npm install -g @angular/cli
```
新建项目
```text
ng new publicAngular --skip-install
```
安装依赖
```text
cd publicAngular
cnpm install
```
